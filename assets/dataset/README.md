# Dataset Folder

This folder contains structured medicine data files for initial database population.

## Files

- `diabetes_medicines.csv` - Sample diabetes medicines dataset

## CSV Format

```
id,name,category,generic_name,dosage_form,strength,dosage,side_effects,contraindications
1,Metformin,Oral Medication,Metformin HCl,Tablet,500mg,500-1000mg twice daily,Nausea...,Kidney disease...
```

## Usage

Import this dataset during initial application setup to populate the medicine database.
